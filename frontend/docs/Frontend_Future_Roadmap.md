# Frontend Future Roadmap

The roadmap defines phase milestones for future releases:

* **Phase 1: Foundation (V1)** - Setup folder trees, config variables, layout wrappers, and design tokens (Completed).
* **Phase 2: Component Library (V2)** - Complete component specifications.
* **Phase 3: Auth Logic (V3)** - Integrate JWT tokens storage with backend.
* **Phase 4: Dashboard implementation (V4)** - Build analytics grids.
* **Phase 5: Inventory & Sales (V5)** - Implement inventory adjustment and orders forms.
* **Phase 6: Warehouse (V6)** - Implement picking waves.
* **Phase 7: GRC, BI, ESG (V7)** - Integrated carbon calculators UI.
