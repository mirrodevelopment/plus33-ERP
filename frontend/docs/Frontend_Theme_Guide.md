# Frontend Theme Guide

This manual defines brand guidelines, colors, transitions, and dark/light modes.

## Color Tokens
- Charcoal Black background: `#141414`
- Sidebar panels: `#181818`
- Primary text: `#F5F5F5`
- Accent Coffee Gold: `#C9A46A`

## Glassmorphism Specs
Cards use glass effect with:
```css
background: rgba(255, 255, 255, 0.06);
border: 1px solid rgba(255, 255, 255, 0.08);
backdrop-filter: blur(20px);
border-radius: 16px;
```
