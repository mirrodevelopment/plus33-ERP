# AI UI Guidelines

This manual defines specifications for conversational AI and predictive visualizations.

## AI Chat Interface
- Displays inside a sliding sidebar panel.
- Message blocks are wrapped in glassmorphic bubbles.
- Supports text recommendations and prompt quick-actions list.

## Digital Twin & Telemetry
- Sensory telemetry is displayed using gauge card components.
- Alarms and warnings are highlighted in gold accent / alert danger colors.
