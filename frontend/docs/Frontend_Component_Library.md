# Frontend Component Library

This guide provides usage instructions and API schemas for all reusable UI components.

## Categorized Components Directory

### Layout Components
* **HeaderComponent:** Exposes header shell search inputs and notification triggers.
* **SidebarComponent:** Holds logo branding and main sidebar navigation menus.
* **FooterComponent:** Prints standard copyright metadata.

### Buttons & Cards
* **ButtonComponent:** Dynamic buttons. Supports primary and secondary types.
* **CardComponent:** Glassmorphic layout panels.

### Data & Forms
* **TableComponent:** Structured grids for inventory listings, payroll logs.
* **SearchComponent:** Form search inputs.

### Modals & Feedback
* **ModalComponent:** Focus overlays.
* **ToastComponent:** Alert cards.
* **ProgressBarComponent:** Progress lines.
* **BadgeComponent:** Status identifiers (e.g. Success, Warning, Danger).
